# Misuratore di distanza a ultrasuoni - Arduino

Progetto Arduino che misura la distanza con sensore HC-SR04 e la mostra su display LCD 16x2.
Viene inoltre allegato il codice dove è necessario l'app Arduino IDE, per invece il kit fisico c'è bisogno di "The Most Complete Starter Kit Mgea 2560 Project" di Elegoo.
